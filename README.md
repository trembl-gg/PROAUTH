## 🔐 ProAuth — advanced authorization system with Telegram 2FA

**ProAuth** is a modern authorization plugin for Minecraft servers (Purpur / Paper / Spigot / Bukkit) that provides a high level of security for player accounts. The plugin supports **two-factor authentication via Telegram**, a flexible command system, and convenient administration tools.

### 🚀 Key features

* 🔑 Player registration and login (`/register`, `/login`)
* 🔄 Password change and account logout
* 🤖 **2FA via Telegram**
  Login confirmation with a code from Telegram
* 🛡️ Protection against unauthorized access
* 📊 Admin commands for managing accounts and sessions
* 💾 Data storage in a database
* ⚙️ Support for Minecraft **1.21**

### 🧑‍💻 Commands

* `/register <password> <repeat>` - account registration
* `/login <password>` - account login
* `/logout` - logout
* `/changepassword <old> <new> <repeat>` - password change
* `/2fa enable|disable|code|status` - two-factor authentication management
* `/proauth` - administrative commands
* `/vanishlogin` - in develop

### 📌 Who is ProAuth suitable for?

* Servers with **offline mode**
* Projects with increased security requirements
* Minecraft servers that use Telegram in their project ecosystem

## 🔐 ProAuth — продвинутая система авторизации с Telegram 2FA

**ProAuth** — это современный плагин авторизации для Minecraft серверов (Purpur / Paper / Spigot / Bukkit), обеспечивающий высокий уровень безопасности аккаунтов игроков. Плагин поддерживает **двухфакторную аутентификацию через Telegram**, гибкую систему команд и удобные инструменты администрирования.

### 🚀 Основные возможности

* 🔑 Регистрация и вход игроков (`/register`, `/login`)
* 🔄 Смена пароля и выход из аккаунта
* 🤖 **2FA через Telegram**
  Подтверждение входа кодом из Telegram
* 🛡️ Защита от несанкционированного доступа
* 📊 Админ-команды для управления аккаунтами и сессиями
* 💾 Хранение данных в базе данных
* ⚙️ Поддержка Minecraft **1.21**

### 🧑‍💻 Команды

* `/register <password> <repeat>` - регистрация аккаунта
* `/login <password>` - вход в аккаунт
* `/logout` - выход
* `/changepassword <old> <new> <repeat>` - смена пароля
* `/2fa enable|disable|code|status` - управление двухфакторной аутентификацией
* `/proauth` - административные команды
* `/vanishlogin` - в разработке

### 📌 Для кого подойдёт ProAuth?

* Серверы с **offline-mode**
* Проекты с повышенными требованиями к безопасности
* Minecraft-сервера, использующие Telegram в экосистеме проекта
